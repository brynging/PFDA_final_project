## Description
My project is a simple character creator that allows the user to create, design, or reimagine any character they'd like, or even recreate themself! The user can change the bangs, hair, skin, top, bottom, and shoes to a variety of different options in every color of the rainbow, including some extras. The folder within src entitled "game_graphics" contains all the graphics that the character creator uses. 

In the future, I'd like to be a little more organized and prepared in the coding and naming of files. There are several occasions where I remembered something exists that could make the code more streamlined and less repetitive... only after I'd put in a large block of code. Additionally, it only features two of the three required functions, though I hope the added complexity of the graphics can help make up for it.

The code itself is easily replicable to use in other projects, or add in additional character design elements like accessories, different eyes, facial features, etc. More colors would also be no trouble to add as well. Despite being a standard character creator, this project is well equipped to be reused or built off of for future works, especially for games like RPGs, simulators, visual novels, as well as any other 2D game that benefits from a customizable character.
