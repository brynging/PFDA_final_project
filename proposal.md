# Character Creator

## Repository
https://github.com/brynging/PFDA_final_project.git

## Description
Program will be an interactive character creator/dress up game that could be reused as a character creation minigame for other projects.

## Features
- Object Selection
	- Players will be able to select an object such as hair, clothes, eyes, or shoes by clicking on the specified object.
- Dynamic mannequin
	- A mannequin will update its appearance depending on the players choices. Each choice will be stored in a variable, and the mannequin will be an object with variables applied that can be updated outside the object.

## Outcomes
Ideal Outcome:
- Ideally, the mannequin has customizable hair (able to change color), eyes (able to change color), skin (able to change color), skin details, tops, bottoms, and socks + shoes.
The player will be able to make choices to update the mannequin's appearance, and see what choice they're making as each option will be visually displayed.

Minimal Viable Outcome:
- Mannequin will have customizable hair, skin, tops, bottoms, and socks + shoes. Color will not be able to change and variety will be limited.
Choices will update the mannequin's appearance, and text will show the player what choice they're making.
